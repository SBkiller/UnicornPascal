# UnicornPascal

Pascal language binding for the [Unicorn emulatorr](http://www.unicorn-engine.org/)
([GitHub](https://github.com/unicorn-engine/unicorn)).

## License

GPL

## Compatibility

OS
: Windows, Linux

Compiler
: Delphi, Free Pascal

## Examples

* `SampleArm64` Emulate ARM64
* `SampleX86` Emulate 16, 32, 64 Bit x86
